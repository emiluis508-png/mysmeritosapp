# MySMéritosApp

Calculadora móvil de méritos. Lista para subir a GitHub e importar en **expo.dev** para generar APK.

## Pasos rápidos

1. Crea un repositorio en GitHub llamado `mysmeritosapp`.
2. Sube todos los archivos de este ZIP tal cual.
3. Entra a https://expo.dev, crea cuenta e **importa** el repo.
4. En el panel de `Builds`, elige **Android → APK** (perfil `preview`).
5. Descarga el enlace del APK que genera EAS.

### Ajustes opcionales
- Cambia el paquete Android en `app.json` → `expo.android.package`.
- Modifica el coeficiente por defecto en `App.js` (estado `coef`).
